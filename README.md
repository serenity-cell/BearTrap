# BearTrap
this is a beginner honeypot built from with mainly boost asio

## what does it do??
sets fake services to listen behind ports and whenever something connects to the ports it sest the ip address of the device accesing it, the fake service hit, and the timestamp. Then, it replies with a faken banner and captures any commands or text attempted to put through the port.
